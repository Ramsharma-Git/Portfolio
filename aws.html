Hosting My Portfolio to AWS S3 Bucket

Amazon S3 (Simple Storage Service) is a cloud-based storage service offered by Amazon Web Services (AWS). It provides developers and businesses with secure, durable, and highly scalable object storage for a wide variety of data types, including documents, images, videos, backups, and application data. We can also host Static Website on it, the steps to do so are as follows :


1. Create an S3 Bucket : Go to your AWS Dashbord and then go to S3 service. Here you have to create a bucket with the same name of your Domain.


In my case i have created a bucket with the name ramsharma.site


Uncheck the Block all public access, and click to create bucket.




2. Upload the Website files : After creating the Bucket, we have to upload the files which containes the website code.






3. Enable Static Website Hosting : Now we have to enable Static Website Hosting by going to the Properties tab and scroll to the bottom, and enable Static Website Hosting



4. Copy the Url of your Website : After enabling the website hosting, AWS will provide you an end point which can be used to browse your website. Simply copy it and Paste to your browser.


You can see that my Portfolio is succesfully hosted on S3.
