Basic Monitoring in Linux

Linux Monitoring Commands

1. Top : Top command is a performance monitoring program that is used frequently by many system administrators to monitor Linux performance and it is available under many Linux/Unix-like operating systems.


There is a one more similar command like Top , Htop. This provides a more advance feature and a colorful utput.


You can simply run dnf install htop -y command to install the htop command.


2. VmStat : VmStat command is used to display statistics of virtual memory, kernel threads, disks, system processes, I/O blocks, interrupts, CPU activity, and much more. You need to install the sysstat package to use the vmstat command. It provides the following output :

[root@server ~]# vmstat
procs -----------memory---------- ---swap-------io---- -system-- ------cpu-----

3. Lsof : The lsof command is used in many Linux/Unix-like systems to display a list of all the open files and the processes. The open files included are disk files, network sockets, pipes, devices, and processes. This is used mainly when a disk cannot be unmounted and displays the error that files are being used or opened. 6. Iotop : iotop is also much similar to top command and htop program, but it has an accounting function to monitor and display real-time Disk I/O and processes. iotop tool is much useful for finding the exact process and high used disk read/writes of the processes. You need to install the iotop package in order to use the iotop command.
